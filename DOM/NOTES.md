# Starter Code

    - <style> tag connect HTML with CSS </style>
    - <script> tag connects HTML with JS</script>

# Window Object

    - Window object represents an open window in a browser. It's browser's object (not JavaScripts's) & is created autometically by browser.
    - It is global object with lots of properties and methods.

# DOM manipulation properties

    - tagName : return text for element notes
    - innerText : return the text content of the element and all its children
    - innerHTML : return the plain text or HTML contents in the element
    - textContent : return the textual content even for hidden elements
