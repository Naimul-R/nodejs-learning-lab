## Fetch API (Application Programming Interface)

- The fetch API provides an interface for fetching (sending/receving) resources.
- It use _Request_ and _Responses_ objects
- The _fetch()_ method is used to fetch a resource (data)

> let promise = fetch(url, [options])

## Understanding Terms

- _AJAX_ is Asynchronous JS & XML
- _JSON_ is JavaScript Object Notation
- _json() method_ : returns a secound promise that resolve with the result of parsing the response body text as JSON. (Input in JSON, output in JS object)

## Request & Response

- HTTP Verbs
- Response Status Code

> HTTP response header also contain details about the responses, such as content type, HTTP status code etc.
